# powerbi-hospital-performance-dashboard
Hospital Performance Dashboard: ER Wait Time & Patient Flow (Power BI)
Healthcare-focused Power BI dashboard analyzing ER wait time, patient flow, and hospital operational performance. Designed to identify bottlenecks, monitor key KPIs, and support data-driven decision-making in a clinical setting.
